# Project rules

- **Never kill or restart the dev server** (`next dev`, port 3001) — no `pkill`, no `kill`, no restarting it yourself. If a restart is needed (e.g. after `.env.local` changes), ask the user to do it and wait.
- **No em dashes** in website copy or code comments. Rewrite with a period, comma, colon, or parentheses instead.
